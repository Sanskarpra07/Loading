# Personal Portfolio (HTML/CSS/JS)

This is a single-page personal portfolio built with vanilla HTML, CSS and JavaScript. Replace placeholder text, images and icons as instructed.

Files:
- `index.html` — main structure with comments before every section.
- `css/style.css` — all styles. Edit CSS variables at the top to re-theme.
- `js/main.js` — all JavaScript for animations and interactions.
- `assets/` —  placeholders (profile, thumbs, CV file).

Styling tips:
- Fonts are loaded from Google Fonts: `Space Grotesk` for headings and `Inter` for body.

Accessibility & Performance:
- Semantic HTML and aria-labels included.
- Images use `loading="lazy"` where possible.
